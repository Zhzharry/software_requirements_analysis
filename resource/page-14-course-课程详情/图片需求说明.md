# 页面：课程详情 `/course/:id`

- **路由**：`/course/:id`（主示例 `1002`）
- **视图**：`src/views/CourseDetailView.vue`
- **图片目录（代码读取此路径）**：`resource/page-14-course-课程详情/`

每个课程 id 最多三张图（`.webp` 或 `.png` 均可）。**新增或移动图片后请重启 `npm run dev`**。

| 文件名（放入本目录） | 页面位置 |
|---------------------|----------|
| `p14-1002-cover-01.webp` 或 `.png` | 课程主封面 |
| `p14-1002-qrcode-01.webp` 或 `.png` | 咨询二维码 |
| `p14-1002-related-01.webp` 或 `.png` | 相关推荐课封面 |

将 `1002` 换成实际课程 id。
