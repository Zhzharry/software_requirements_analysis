# 共用 UI 图片需求（多页复用）

以下资源被 **`SiteHeader.vue`**、**`ItheimaWwwPageChrome.vue`** 等组件引用，换图后影响**所有挂载这些组件的页面**。

**统一命名**：`p00-{组件简写}-{用途}-{序号}.{扩展名}`，扩展名建议 **`.webp`** 或 **`.png`**（需透明用 PNG）。

| 序号 | 用途说明 | 建议规格 | 命名示例 | 代码位置（替换时） |
|------|----------|----------|----------|---------------------|
| 1 | 视频库顶栏品牌 Logo（白底条左侧） | 约 **280×96** px（@2x），显示宽约 140px | `p00-sh-logo-01.webp` | `src/components/SiteHeader.vue` → `PlaceholderImage` `ratio="140 / 48"` |
| 2 | 顶栏用户头像占位 | **56×56** px 方图（@2x），圆角由 CSS 裁切 | `p00-sh-user-01.webp` | `SiteHeader.vue` → `ratio="1"` `site-head__user-ico` |
| 3 | 官网壳（师资/开班/学费/就业）顶栏 Logo | 约 **320×96** px（@2x） | `p00-iwc-logo-01.webp` | `ItheimaWwwPageChrome.vue` → `ratio="160 / 48"` |

**说明**：项目根已有 `resource/logo.png`，可将 **(1)(3)** 统一为同一母版导出不同宽度，命名仍建议按上表区分副本以便缓存与替换。
