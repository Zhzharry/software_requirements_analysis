# 页面：学习路线图 `/subject/aiappmap`

- **路由**：`/subject/aiappmap`
- **视图**：`src/views/SubjectAiappmapView.vue`
- **图片目录（代码读取此路径）**：`resource/page-16-subject-aiappmap-学习路线图/`

当前页仅在**页头品牌行**使用一张可选自定义图；**无图时**使用全局 `BrandLogo`。

支持 **`.png` / `.webp` / `.jpg`**。`pageImage` 会按扩展名自动匹配。新增或改名后请**重启 `npm run dev`**。

## 文件名（任选其一，放入本目录）

按优先级从高到低尝试加载，**只需放一个文件**：

| 建议文件名 | 说明 |
|------------|------|
| `p16-head-logo-01.png` 或 `.webp` | **推荐** |
| `p16-head-logo.png` | 无 `-01` 序号也可 |
| `p16-logo-01.png` / `p16-logo.png` | 简写名 |
| `logo.png` | 通用名 |

以上任一对上即可在页头显示；均无时显示默认 `BrandLogo`。
