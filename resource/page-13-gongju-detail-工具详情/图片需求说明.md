# 页面：学习工具详情 `/gongju/:id`

- **路由**：`/gongju/:id`
- **视图**：`src/views/GongjuDetailView.vue`
- **图片目录**：`白马程序员内部/resource/page-13-gongju-detail-工具详情/`

每个工具 id 需要两张图（示例 id=105）：

| 文件名（放入本目录） | 页面位置 |
|---------------------|----------|
| `p13-105-icon-01.webp` | 工具图标 |
| `p13-105-course-01.webp` | 关联课程封面 |

将 `105` 换成实际 `route.params.id`（101、102、103、104、106 等）。
