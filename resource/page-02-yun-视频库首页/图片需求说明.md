# 页面：视频库首页 `/yun`

- **路由**：`/yun`
- **视图**：`src/views/HomePrototype.vue`
- **图片目录（请把文件放在本文件夹）**：`白马程序员内部/resource/page-02-yun-视频库首页/`
- **共用顶栏 Logo**：`../00-shared-global-ui/logo.png`（代码已接，无需放本目录）

支持扩展名：`.webp`、`.png`、`.jpg`、`.jpeg`。列表项可只放 `-01`，其余序号会自动回退到 `-01`。

**命名规则（本页）**：`p02` = 第 02 页（视频库）→ `-` → 用途英文小写（如 `hero-banner`、`float`）→ `-` → 两位序号 `01` → **一个**扩展名。不要用 `image` 代替序号，也不要写成 `xxx.png.webp`。

| 文件名（放入本目录） | 页面位置 |
|---------------------|----------|
| `p02-hero-banner-01.webp` 或 `p02-hero-banner-01.png` | 首屏 Banner 大图（二选一扩展名即可） |
| `p02-float-01.webp` | 右侧悬浮入口 1 |
| `p02-float-02.webp` | 右侧悬浮入口 2 |
| `p02-float-03.webp` | 右侧悬浮入口 3 |
| `p02-promo-01.webp` | 运营卡片 1 |
| `p02-promo-02.webp` | 运营卡片 2 |
| `p02-promo-03.webp` | 运营卡片 3 |
| `p02-promo-04.webp` | 运营卡片 4 |
| `p02-live-01.webp` … `p02-live-08.webp` | 直播公开课封面（8 条，可只放 01） |
| `p02-free-01.webp` … `p02-free-06.webp` | 免费课程封面（6 条，可只放 01） |
| `p02-road-01.webp` … `p02-road-09.webp` | 学习路线图卡片（9 条，可只放 01） |
